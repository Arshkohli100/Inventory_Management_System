## Quick Installation

    cd Inventory_Management_System
    
### Composer

    composer update
    
    
### For Environment Variable Create
 
    cp .env.example .env
 
    
 ### For Migration table in database [Create database name as ```IMS```]
 
    php artisan migrate
    
### To Run the server

    php artisan serve
